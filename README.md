# Turing Machines and Pushdown Automata - Computational Models Project

## Project Overview

This project is a comprehensive study and implementation of Turing Machines and Pushdown Automata within computational models. It combines theoretical analysis with practical programming implementations.

## Project Structure

### Part 1: Comprehensive Report

A detailed report covering:
- Practical implementations of Turing Machines and Pushdown Automata
- Advantages and disadvantages analysis
- Types of problems they address
- Limitations and drawbacks
- Supporting insights and additional research



### Part 2: Programming Implementation

A Java/C++/C# application with GUI implementing:

1. **CFG to PDA Conversion** - Convert Context-Free Grammar to Pushdown Automaton
2. **DFA Implementation** - Construct a DFA over {0,1} that accepts strings where the number of 1s is divisible by 3 and ends with 0
3. **PDA for aⁿbⁿ Language** - Implement a Pushdown Automaton for L = { aⁿbⁿ | n ≥ 0 }
