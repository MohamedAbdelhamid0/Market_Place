# Marketplace

Two-sided e-commerce platform with buyer and seller dashboards.



## Tech Stack

- HTML5, CSS3, JavaScript
- localStorage (browser storage)
- Responsive dark UI

## Features

**Buyers**
- Browse products
- Search products
- Place orders
- Track order status
- Write reviews

**Sellers**
- Add products
- Manage inventory
- Track orders
- Update order status
- Manage profile

## Files

- `auth.js` - Authentication & session
- `app.js` - Seller dashboard
- `buyer2.js` - Buyer logic
- `Buyer.html` - Buyer page
- `seller.html` - Seller page
- `buyerLogin.html` - Buyer login
- `sellerLogin.html` - Seller login


## Data

All data stored locally in browser (localStorage). No server needed for Phase 1.
